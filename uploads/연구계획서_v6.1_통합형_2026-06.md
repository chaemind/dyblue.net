# Ⅰ. 연구자의 출발점

## 1. 정책이 주민에게 도달하는 과정에 대한 관심

나의 연구는 사용자경험과 서비스디자인에 대한 관심에서 출발하였다. 초기 관심은 시민이 공공서비스와 정책을 실제로 어떻게 경험하는지, 정책과 제도가 주민의 삶에 도달하는 과정에서 어떤 단절이 발생하는지에 있었다.

지방의회 정책지원관으로 근무하면서 관심은 서비스의 설계 자체보다 다음의 제도적 과정으로 이동하였다.

- 주민의 요구와 현장정보가 어떻게 의제로 번역되는가
- 상충하는 이해관계와 지식이 어디에서 조정되는가
- 주민참여와 숙의의 결과가 조례·예산·행정사무감사로 연결되는가
- 정책 집행 이후의 결과와 부작용이 다시 제도수정으로 환류되는가
- 의원·정책지원관·의회사무기구·집행부는 이 과정에서 어떤 역할을 수행하는가

따라서 서비스디자인은 학위논문의 상위 이론이 아니라 **연구문제가 형성된 배경이자 실천방법의 하나**로 위치시킨다.

## 2. 연구자로서의 실무적 위치

지방의회 정책지원 업무는 다음 자료와 과정에 대한 직접적인 관찰기회를 제공한다.

- 조례 제·개정 과정
- 예산·결산 심사
- 행정사무감사와 자료요구
- 민원·청원·건의의 의제화
- 의원·주민·집행부 간 정보와 요구의 이동
- 의정활동 이후의 후속조치와 제도개선

이 실무경험은 연구대상에 대한 접근성을 높이지만, 동시에 내부자 연구자의 편향과 연구윤리 문제를 발생시킬 수 있다. 따라서 공개자료, 비식별 자료, 연구참여 동의, IRB 검토, 연구자 성찰기록을 구분하여 활용한다.

# Ⅱ. 변하지 않는 연구축

다음 다섯 가지는 탐색단계에서도 변경하지 않는다.

| 고정 항목 | 내용 |
|---|---|
| **연구대상** | 지방의회: 집행기관이 아닌 주민대표기관 |
| **문제상황** | 복잡성·불확실성·상호의존성을 가진 지역문제 |
| **핵심과정** | 감지 → 지식통합 → 숙의 → 제도화 → 재점검·환류 |
| **규범적 조건** | 대표성 · 책임성 · 투명성 · 검토가능성 |
| **주요 행위자** | 의원 · 의회사무기구 · 정책지원관 · 주민 · 집행부 |

## 1. 상위 연구문제

> **복잡하고 불확실한 지역문제에서 지방의회는 주민·이해관계자·집행부에 분산된 요구와 지식을 어떻게 감지·통합·숙의하고, 이를 조례·예산·행정사무감사와 후속 제도수정으로 전환하는가?**

이 상위 질문에는 다음 하위 문제가 결합된다.

- 전환과정을 가능하게 하는 개인적·조직적·제도적 조건은 무엇인가?
- 지방의원과 정책지원관은 각각 어떤 역할을 수행하는가?
- 퍼실리테이션과 정치적 경계확장은 어떤 작동 메커니즘인가?
- 서비스디자인·코디자인·공론장은 어떤 상황에서 사용되는 실천방법인가?
- AI는 지식통합·숙의·검증을 지원하는가, 소수의견·갈등·책임소재를 약화하는가?
- 지방의회의 대표성·책임성·공식 권한은 일반 협력적 거버넌스 네트워크와 무엇이 다른가?

# Ⅲ. 왜 지방의회인가

지방의회가 어댑티브 거버넌스를 자동으로 수행한다고 전제하지 않는다. 지방의회는 다음 특성으로 인해 어댑티브 거버넌스의 작동 가능성과 실패조건을 함께 관찰할 수 있는 **대표제적 제도 장**이다.

| 지방의회의 제도적 특질 | 검토할 거버넌스 기능 |
|---|---|
| 근접성·현장성 | 주민 요구와 정책결과의 변화 감지 |
| 다중 행위자 접점 | 주민·집행부·전문가·이해관계자 간 지식 연결 |
| 대표성과 정치적 경쟁 | 다원적 관점의 표출과 갈등의 제도화 |
| 조례·예산·감사권한 | 숙의결과를 공식 정책수단으로 전환 |
| 반복적 의정과정 | 정책결과의 점검과 재의제화 |
| 다중심적 지방제도 | 지역별 정책실험과 상호학습 가능성 |

> **핵심 단서:** 이러한 구조는 잠재적 조건일 뿐이며 실제 작동은 검증대상이다. 본 연구의 질문은 지방의회가 어댑티브한가가 아니라, **어떤 조건에서 감지·학습·제도수정이 발생하거나 중단되는가**이다.

# Ⅳ. 이론적 출발점과 현재의 불확실성

## 1. 어댑티브 거버넌스 작업정의

어댑티브 거버넌스(adaptive governance, 이하 AG)란 복잡성과 불확실성으로 정책결과를 완전히 예측하기 어려운 상황에서, 다양한 행위자와 지식을 연결하고 정책과 제도의 결과를 관찰·학습하여 규칙·역할·권한·책임을 수정하는 거버넌스 과정과 제도적 역량을 의미한다.

| 요소 | 내용 |
|---|---|
| 문제조건 | 복잡성 · 불확실성 · 변화 · 상호의존성 |
| 행위구조 | 다층적 행위자 · 분산된 지식 · 상충하는 이해관계 |
| 작동과정 | 감지 · 해석 · 숙의 · 학습 · 실험 · 피드백 |
| 제도적 결과 | 정책수단 · 규칙 · 역할 · 권한 · 책임의 수정 |

AG는 단순한 신속성·유연성·참여확대와 동일하지 않다. 새로운 정보와 정책결과가 실제 제도수정으로 연결되는지가 핵심이다.

## 2. G1: AG의 이론적 위치 판정

AG를 학위논문의 확정된 상위이론으로 전제하지 않는다. 2026∼2027년 이론검토와 예비사례 분석을 통해 다음 세 위치 중 하나로 판정한다.

| 판정 | 위치 | 판단기준 |
|---|---|---|
| **G1-A 핵심 설명이론** | 지방의회의 행위와 결과를 독자적으로 설명 | 작동 메커니즘, 경쟁이론 대비 추가 설명력, 실패조건이 확인됨 |
| **G1-B 중범위 분석틀** | 사례와 정책과정을 구조화하는 분석범주 | 감지·학습·환류가 유용하나 독자적인 인과명제는 제한됨 |
| **G1-C 문제설정 프레임** | 복잡성과 불확실성을 제시하는 배경 | 현상을 정당화·동기화하지만 직접적인 관찰·설명모형으로 연결되지 않음 |

G1은 다음 세 질문으로 판정한다.

1. 어떤 조건에서 감지·학습·제도수정이 발생하는가?
2. 정책역량·협력적 거버넌스·정치적 경계확장보다 무엇을 더 설명하는가?
3. AG가 작동하지 않았다고 판단할 수 있는 실패조건은 무엇인가?

# Ⅴ. 후보 연구경로와 층위 배치

학위논문은 후보 중 하나를 즉시 선택하는 문제가 아니라, 각 개념을 어느 층위에 배치하고 무엇을 실증할 것인지 결정하는 문제이다.

```text
장기 연구프로그램 C4
지방의회의 적응적 거버넌스가 인간적·제도적·기술적 조건에서 어떻게 작동하는가
        │
        ├─ G1 판정: AG의 위치
        │   ├─ 핵심 설명이론
        │   ├─ 중범위 분석틀
        │   └─ 문제설정 프레임
        │
        ├─ C2 과정·제도 경로
        │   감지–지식통합–숙의–제도화–환류의 과정모형
        │
        ├─ C1 행동역량 경로
        │   의원 또는 의원–지원인력의 퍼실리테이팅 행동역량
        │
        └─ C3 AI 매개 경로
            지식통합·숙의·검증·책임성에 대한 사회기술적 조건
```

## 1. C1 — 퍼실리테이팅 행동역량

지방의원 또는 의원–정책지원인력의 다음 행동이 독립적 구성개념을 형성하는지 검토한다.

- 현장정보와 주민요구의 의제 번역
- 상충하는 지식과 이해관계의 연결
- 갈등조정
- 참여·숙의 절차의 구조화
- 조례·예산·감사로의 공식화
- 정책결과의 재점검과 재의제화

**핵심 위험:** 기존 퍼실리테이브 리더십, 네트워크 거버넌스 리더십, 정치적 경계확장, transversal leadership과의 중복.

## 2. C2 — 지방의회 과정·제도 경로

AG를 단일 심리척도로 측정하지 않고, 지방의회가 정책신호를 감지하고 제도변화로 연결하는 과정을 분석한다.

잠정 과정:

> 감지 → 해석·의제번역 → 지식통합 → 숙의·갈등조정 → 공식 의정활동 → 집행결과 점검 → 재의제화·제도수정

이 경로는 현재 가장 유력한 과정연구 후보이지만 G1 판정 전까지 확정하지 않는다.

## 3. C3 — AI 매개 지방의회

AI를 기술수용이나 업무효율만으로 다루지 않고 다음 민주적·인식론적 결과와 연결한다.

- 근거연계성과 출처투명성
- 소수의견의 보존
- 의견요약과 갈등의 평탄화
- 검증행동과 자동화 의존
- 인간의 판단재량
- 책임귀속과 기록가능성
- 공식 의정절차와의 연결

도전학기와 소논문 B의 결과를 토대로 2027년 말 학위논문 편입 여부를 결정한다.

## 4. C4 — 장기 통합 연구프로그램

C4는 학위논문의 단일 실증모형이 아니다.

- AG: 이론적·분석적 기반
- 퍼실리테이션: 인간적 작동 메커니즘
- 서비스디자인·코디자인·공론장: 실천방법
- AI: 사회기술적 조건
- 의정지원체계·제도·외부관계: 후속 연구 층위

- C2는 지방의회 고유성을 설명할 가능성이 높지만 단일 척도로 측정해서는 안 됨.
- C3는 독창성과 확장성이 높지만 기술변화·윤리·자료 접근 위험이 큼.
- C4는 장기 연구정체성으로는 적합하나 학위논문의 단일 실증경로로는 부적합함.

# Ⅶ. 구성개념과 연구설계의 핵심쟁점

## 1. 판별타당도는 통계 이전에 이론에서 시작함

| 판별영역 | 확인할 질문 |
|---|---|
| **개념** | 기존 경계확장·리더십보다 무엇을 추가하는가 |
| **행위주체** | 공공관리자가 아닌 선출직 의원을 연구해야 하는 이유는 무엇인가 |
| **제도맥락** | 대표권·조례·예산·감사권한이 어떤 차이를 만드는가 |
| **핵심행동** | 협업 촉진을 넘어 공식 의제화와 제도환류가 확인되는가 |
| **결과변수** | 협력·신뢰 외에 정책수정·재의제화·책임성을 설명하는가 |
| **분석수준** | 개인역량·집합역량·조직조건·제도조건을 구분할 수 있는가 |

HTMT, Fornell–Larcker, 증분타당도는 이론적 판별 이후의 검증수단이다.

## 2. 반영·형성·과정·다층모형

의제화·지식연결·갈등조정·절차설계·제도환류를 다차원 반영척도로 미리 확정하지 않는다.

| 모형 | 의미 |
|---|---|
| **반영모형** | 하나의 잠재적 역량이 여러 행동에 공통으로 반영됨 |
| **형성모형** | 서로 다른 기능들이 결합하여 상위 역량을 형성함 |
| **과정모형** | 감지부터 제도환류까지 시간적·인과적으로 연결됨 |
| **다층모형** | 개인·지원체계·제도·외부관계가 서로 다른 수준에서 작용함 |

Grøn과 Jacobsen의 transversal leadership 척도는 세 차원으로 이론화되었으나 차원 간 판별타당도가 확보되지 않아 실질적인 1차원 구성개념으로 처리되었다. 이는 본 연구의 잠정 차원이 붕괴한다는 직접 증거가 아니라, 측정구조를 사전에 확정하지 말아야 한다는 경고로 활용한다.

## 3. C1 판정체계

### C1 유지

다음 조건 중 세 가지 이상을 충족할 경우 독립적 구성개념 또는 구성개념 확장 가능성을 유지한다.

1. 기존 경쟁개념에 없는 지방의회 특유 행동이 반복 확인됨.
2. 주민·현장정보의 공식 의제화 행동이 확인됨.
3. 조례·예산·감사와 후속점검을 연결하는 행동이 확인됨.
4. 일반적 성격이나 리더십이 아니라 역할수행 능력으로 구별됨.

### C1 수정 A — 맥락 확장

기존 정치적 경계확장·transversal leadership과 행동은 유사하지만, 선출직 대표성·공식 권한·지방의회 맥락이 추가되는 경우.

가능한 연구:

- 정치적 경계확장의 지방의회 적용
- 지방의회형 퍼실리테이브 리더십
- 기존 척도의 지방의원 적용과 측정동일성 검토

### C1 수정 B — 측정모형 전환

다음 결과는 중단이 아니라 모형전환 사유로 처리한다.

- 잠정 하위차원이 하나의 요인으로 수렴
- 요소들이 상위 역량을 형성
- 요소들이 시간적 과정단계로 나타남
- 의원 개인보다 의원–정책지원관–사무기구의 집합적 역할이 중요함
- 개인·조직·제도 수준의 다층구조가 요구됨

### C1 중단

다음 두 조건을 모두 충족할 경우 독립적인 지방의원 퍼실리테이팅 역량 척도개발을 중단한다.

1. 관찰행동과 결과가 기존 경쟁개념으로 충분히 설명됨.
2. 지방의회 맥락에서 추가할 고유 행동영역·결과변수·분석수준이 확인되지 않음.

자료와 표본 확보의 어려움은 이론적 중단이 아니라 연구설계 변경사유로 별도 관리한다.

# Ⅷ. 경험적 연구설계 후보

## 1. 내재적 단일사례설계

C2 경로를 추진하는 경우 하나의 지방의회를 주 사례로 설정하고, 복수의 정책과정을 하위 분석단위로 설정하는 **내재적 단일사례설계(single embedded case study design)**를 우선 검토한다.

### 예비 단계

- 2026년 여름: 비식별 사례 스케치 3건
- 2026년 하반기: 공개자료 기반 사례 후보 3∼6건 목록화
- IRB·자료접근성 확인 후: 정식 사례와 인터뷰 범위 확정

### 사례유형

- 주민 요구가 제도수정으로 연결된 사례
- 의정활동은 있었으나 후속 환류가 중단된 사례
- 갈등·정보가 존재했으나 공식 의제가 되지 못한 사례
- 행정사무감사 후 제도개선이 이루어진 사례
- 정책결과가 재검토되어 조례·예산이 변경된 사례

## 2. 조건부 척도개발

G2·G3에서 C1이 유지 또는 수정 A로 판정되는 경우에만 추진한다.

- 질적 문항생성
- 내용타당도
- 인지면접
- EFA
- 독립 표본 CFA
- 기존 경쟁척도와 판별·증분타당도
- 외부 준거변수와의 관계

## 3. 조건부 AI 실험

C3가 유력해지는 경우 제품명이 아닌 시스템 기능을 조작한다.

- 일반 생성형 AI
- 공식 근거자료 연계형 시스템
- 출처투명성
- 반대·소수의견 보존
- 검증가능성
- 책임귀속

종속변수는 활용의향이나 만족도에 한정하지 않고 인식론적 품질·검증행동·재량·책임성으로 확장한다.

# ⅩⅢ. 기대기여

## 1. 지방의회 연구

- 의정활동 건수나 출석률을 넘어 주민요구가 공식 제도로 전환되는 과정을 분석함.

## 2. 거버넌스 이론

- 사회생태계·집행기관 중심 AG 연구를 대표기관인 지방의회로 확장할 가능성을 검토함.
- 대표성·책임성·공식 권한을 AG의 조건과 한계에 포함함.
- 협력적 거버넌스와 대의민주주의의 접점을 분석함.

## 3. 구성개념과 방법론

- 퍼실리테이팅 역량과 기존 리더십·경계확장 개념의 경계를 검토함.
- 반영·형성·과정·다층모형을 경쟁적으로 비교함.
- 구성개념 개발 이전에 이론적 판별과 분석수준을 확인하는 연구절차를 제시함.

## 4. 실무적 기여

- 주민요구가 의정활동과 제도수정으로 연결되는 과정의 단절지점을 진단함.
- 지방의원·정책지원관·의회사무기구의 역할과 지원조건을 구체화함.
- 향후 지방의회 역량개발·교육·진단도구의 근거를 마련함.

## 5. AI와 의회 민주주의

- 인간–AI 협업형 의정지원의 제도조건을 검토함.

# 연구자로서 사용할 핵심 문장

나는 AG를 단순한 유연성이나 시민참여 확대가 아니라, 복잡성과 불확실성 속에서 새로운 정보와 정책결과를 감지·학습하고 이를 규칙·권한·책임의 수정으로 연결하는 거버넌스 과정으로 이해한다.

지방의회는 이러한 과정을 자동으로 수행하는 기관이 아니라, 대표성·조례·예산·감사라는 공식 권한을 통해 분산된 요구와 지식을 제도변화로 전환할 가능성과 한계를 동시에 가진 대표제적 제도 장이다.

퍼실리테이션은 AG의 상위 개념이 아니라 이를 작동시키는 메커니즘 후보이며, 지방의원 퍼실리테이팅 역량의 독립성은 기존 리더십·경계확장 개념과의 이론적 판별을 거쳐야 한다.
